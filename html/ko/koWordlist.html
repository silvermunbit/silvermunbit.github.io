Particles

조사

The Japanese equivalent is also given.

  • 는/은 topic は
  • 가/이 subject が. 나 → 내가, 저 → 제가, 너 → 네가, 누구 → 누가.
  • 를/을 direct object を
  • 도 also も
  • 만 only だけ. ~에만, ~때만
  • 에 place/direction, indirect object (inanimate) に
  • 의 possessive の. 나의/내, 저의/제, 너의/네, 누구의/누구.
  • 에서 from (place) から, place で
  • 부터 from (time) から
  • 까지 until まで
  • 로/으로 (ㄹ로) instrumental で, direction へ. 아침식사로 for breakfast, 두 번째로 for the second time, 처음으로 for the first time.
  • 와/과, 랑/이랑, 하고 comitative と
  • 에게, 한테, 께 indirect object (animate) に
  • 에게서, 한테서, 로/으로부터 from (person) に
  • 보다 than より

Adverbs

부사

Irregular adverbs from adjectives.

  • 빠르다 → 빨리 quickly
  • 많다 → 많이 many
  • 높다 → 높이 highly, up high
  • 깊다 → 깊이 deeply
  • 같다 → 같이 together, like
  • 가깝다 → 가까이 near
  • 멀다 → 멀리 far
  • 깨끗하다 → 깨끗이 cleanly
